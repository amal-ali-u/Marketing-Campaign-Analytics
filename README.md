
# Marketing Campaign Analytics
This project analyzes marketing campaign data of 2,240 customers from Maven Marketing.  
It includes Galaxy Schema design, SQL Data Warehouse, ETL workflows with SSIS, reporting with SSRS, and interactive dashboards in Power BI.  

## Key Features
- Customer segmentation (Regular vs High Spenders)
- Product preferences (Wines, Meat, Gold, etc.)
- Campaign effectiveness (responses vs non-responses)
- Channel performance (Store, Web, Catalog)

## Technologies Used
-Excel_ power query
- SQL Server Data Warehouse
- SSIS for ETL
- SSRS for reporting
- Power BI for visualization
